# Portail Partenaire Mobile

Utilise la version de node 16.19.0

## Liste des commandes utiles et courante
```
npm install
```

### Compiles and hot-reloads for development
```
ionic serve
```

### Compiles and minifies for production
```
ionic build
```

### Capacitor add platforms
```
ionic cap add ios
ionic cap add android
```

### Capacitor copy the changes to native folders after build 
```
ionic cap copy
```

### Capacitor sync after adding a native element (such as plugin)
```
ionic cap sync
```

### Capacitor open project on native IDE (Android studio/XCode) in order to generate the packages
```
ionic cap open ios
ionic cap open android
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
